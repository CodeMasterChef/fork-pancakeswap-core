# fork-pancakeswap-core:
This is PancakeSwap Factory which forked from https://github.com/pancakeswap/pancake-swap-core.
# Local Development:
- Node>=10.
- Truffle Suite (https://www.trufflesuite.com/)
## Install Dependencies:
```
npm install
```
## Compile Contracts:

```
npm run compile
```

## Migrate Contracts:
Local:
```
npm run migrate
```
BSC Testnet:
```
npm run migrate:testnet
```
## Run Tests:
```
npm run test
```
